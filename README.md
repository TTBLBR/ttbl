# TTBL Front

Front page da TTBL criada em Next.js, sem Tailwind e sem Supabase.

## Páginas já preparadas

- `/rio-zero-um`
- `/calendarios`
- `/categorias`
- `/regras`
- `/quem-somos`
- `/ranking`

Cada bloco da página inicial abre uma dessas páginas internas para edição futura.

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy na Vercel

- Root Directory: vazio
- Install Command: `npm install`
- Build Command: `npm run build`
- Output Directory: vazio
